flourthprinter

Flourthprinter is a running project I created with my roomate, Adam, after purchasing an NCR 7167 thermal printer from eBay and jerryrigging it to work with a Raspberry Pi.

Any tweet sent that contains @FlourthPrinter will immediately print out in our dorm room.

The code is 100% written in python, and runs on a Raspberry Pi Zero W. Here is a link to the twitter page.
