verifried

verifried is an Android app created for a CS125 final project. The idea was an app that would only let the user tweet at verified accounts. My partner and I had a few weeks to learn UI design and the Twitter API (see flourthprinter). The final product worked great, and was a fun exercise in collaboration and UI design.
